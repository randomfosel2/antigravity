# Instagram Influencer Analysis

## 프로젝트 소개
전 세계 상위 인스타그램 인플루언서 200명의 데이터를 기반으로, 인플루언서의 팔로워 수, 좋아요 수, 참여율(Engagement Rate), 영향력 점수 등을 분석하고 시각화하는 프로젝트입니다.

## 데이터
- `data/insta_influencers_tableau_ready.csv`

## 주요 분석 내용 (예정)
- 국가별/언어별 인플루언서 분포
- 팔로워 수와 참여율의 상관관계 분석
- 영향력이 높은 인플루언서들의 공통적 특징 도출
