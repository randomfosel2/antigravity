# World Press Freedom Index Dashboard

## 프로젝트 소개
세계 언론 자유 지수(World Press Freedom Index) 데이터를 바탕으로 국가별 언론의 자유도를 시각화한 프로젝트입니다. Tableau를 활용하여 대시보드를 구성하였습니다.

## 데이터
- `data/World_Press_Freedom_Index.csv`

## 대시보드 파일
- `dashboard/world_press_freedom_index.twb`

## 주요 분석 내용
- 국가별 언론 자유도 순위 현황 및 맵 시각화
- 연도별 언론 자유 지수 변화 트렌드 (있는 경우)
